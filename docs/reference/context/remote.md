# Context Remote

Represents a user who is connected to Bifröst.

## Properties

<<property("host", "Host", "../data-type.md#host")>>

Host of the remote machine.

<<property("user", "string")>>

Host of the requested remote user. Usually: `<user>@my-great-domain.tld`
